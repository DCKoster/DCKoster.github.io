Strategy

One solution to the problem of having infinite worlds is given by \citeauthor{vanditmarschWhoHasSum2015a} \citeyearpar{vanditmarschWhoHasSum2015a} and is described in this section. We will copy their strategy, but while \citeauthor{vanditmarschWhoHasSum2015a} do not mention Kripke models, we will explain the strategy more formally as a Kripke model, including and public announcement logic.

Even though \citeauthor{vanditmarschWhoHasSum2015a} do not mention Kripke models, they do create one, instead calling it a binary tree of uncertainty. This model is constructed by starting with world \( (2, 1, 1) \), then adding all worlds that have a relation with this world, then adding all worlds that have a relation with those worlds, and so on. Every such recursive step is done in the following, systematic way.

Each world has 3 relations \( \sim_a \), \( \sim_b \), and \( \sim_c \) with other worlds, 1 for each agent, because as explained before, the agents know they have either the sum or the difference of the other two numbers. We claimed that each world has 3 relations to other worlds, but there is an exception: the world \( (2, 1, 1) \). We call this a "root world", because this is starting point from where all worlds are added to the model with equation \ref{eq:branch}. We define a root world as follows:

A root world is a world that has a smaller number of relations with other worlds than the number of agents, before any public announcements are made.(2)

Specifically,

For the riddle “Who has the Sum” with 3 agents, a world is a root world iff the difference between the two smallest numbers is smaller than or equal to 0.(3)

In a root world, at least one of the agents knows what their own number is before any announcements are made. For example, the root world \( (2, 1, 1) \) has only two relations because Achmed sees two 1’s, and therefore knows he himself has the sum 2, since the difference \( 1 − 1 \) is 0, and only positive integers are allowed according to the riddle.

As mentioned, public announcement act as model transformers. When an announcement \( \varphi \) is made, all worlds where \( \varphi \) does not hold, are removed from the model to form the reduced model. Now consider the case where there are no root worlds. All worlds have three relations with other worlds, and therefore in every world, every agent is not sure what their own number is. In this case, when any of the first three announcements are made, the reduced model is the exact same as the initial model, because in every world it holds that none of the agents know their number. Additionally, Achmeds announcement \( K_a 50_a \) would create an empty reduced model.

This makes it clear that the riddle can not be solved when there are no root worlds. When there are root worlds however, these may not be included in the reduced model after an announcement is made. This makes it so that the worlds which had relation \( \sim_i \) with the root world, agent i now knows what their number is. The riddle can therefore be solved by reducing the model four times, once for every public announcement, if it eventually leads to a reduced model where Achmed knows that his number is 50. The answer to the riddle would be the numbers of Bert and Carl in that world.

This also solves the problem of having infinite worlds. For the same reason that a model without any root world r would not lead to a solution, the worlds w where k > 4 in r →k w, are irrelevant. This is because it is impossible for any agent to know their number in any of these worlds even after all announcements are made. Additionally, none of the worlds where k = 4 can be worlds where Achmed knows his number is 50. These worlds are only necessary to model because they can have relations with worlds where k = 3, which can be worlds where Achmed knows his number is 50.
